## How to run The Big Dipper

1. Copy `settings.json.default` to `settings.json`.
2. Update paths to `gaiacli` and `gaiadebug`.
3. Update the RPC and LCD URLs.

```
meteor npm install
meteor update
meteor --settings settings.json
```
